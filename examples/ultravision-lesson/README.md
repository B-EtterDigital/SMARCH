# UltraVision lesson fixture

A minimal, self-contained SUP (`.UltraVision`) plan holding the single task that
[intro lesson 17](../../docs/intro/17-reading-the-plan-uvp.md) reads. It exists
so the lesson can demonstrate reading a machine-readable plan without shipping a
project's real, private planning tree. It is a teaching copy, not a live plan —
nothing here is claimed, completed, or verified against real work.
