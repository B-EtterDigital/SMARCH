# 05 — Media Plan

**Classification verdict:** `modules.json.product.media_class = "utility"`.
SMARCH is a CLI framework and self-hostable dashboard; it has no generative
media pipeline, no style bible, and no paid asset batches. Per SMA-C7, no
media/paid tasks exist in this plan.

The only media-adjacent work is the V-13 demo: a scripted terminal recording
(charmbracelet `vhs` tape → GIF/MP4: scan → brick wall → two agents collide →
conflict surfaces gracefully) plus dashboard screenshots. These are ordinary
`docs`-domain tasks (UV-DO-demo-*), unpaid, reproducible from the committed
tape script — not a media pipeline.
