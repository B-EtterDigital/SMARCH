# Support

Start with the [beginner introduction](docs/intro/START_HERE.md). If a SMARCH
term is unfamiliar, use the [glossary](docs/GLOSSARY.md).

- **Questions and discussions:** search the existing
  [`question` issues](https://github.com/B-EtterDigital/SMARCH/issues?q=is%3Aissue+label%3Aquestion),
  then [start a question](https://github.com/B-EtterDigital/SMARCH/issues/new?labels=question&title=Question%3A+)
  if yours is new. GitHub Discussions is not enabled for this repository, so
  this is the live discussion route.
- **Bugs:** search [open bugs](https://github.com/B-EtterDigital/SMARCH/issues?q=is%3Aissue+is%3Aopen+label%3Abug),
  then [open a pre-labelled bug report](https://github.com/B-EtterDigital/SMARCH/issues/new?labels=bug&title=Bug%3A+).
  Include what you ran, what you expected, what happened, and the smallest safe
  reproduction you can share.
- **Vulnerabilities:** use the repository Security tab to
  [report a vulnerability privately](https://github.com/B-EtterDigital/SMARCH/security/advisories/new).
  Do not open a public issue; see [SECURITY.md](SECURITY.md).

Please follow the [Code of Conduct](CODE_OF_CONDUCT.md) wherever you ask for help.
