# F5-UltraVisionPlan: [SUP](../../docs/GLOSSARY.md#sup)

This README explains the opt-in planning skill that reconstructs a product vision and expands it into an auditable UltraVision task graph. Planning leads and execution controllers need it when a request explicitly calls for maximum-granularity planning or an existing `.UltraVision/` plan must continue. Read it before invoking the skill or handing its generated tasks to another agent. Remember that JSONL task records are the source of truth and Markdown task lists are generated views.

See [SKILL.md](./SKILL.md) for the full pipeline, gates, and `uvp` command contract.
