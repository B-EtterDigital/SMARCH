# SMA Course Builder

This README explains the skill that turns registry data into brick wiki pages and a guided learning course. Documentation maintainers and educators need it when they publish or refresh human-facing Sweetspot material. Read it before generating the wiki or course from the global registry. Remember to refresh registry data first so the teaching material reflects current bricks and gates.

See [SKILL.md](./SKILL.md) for generation commands, teaching rules, and reference material.
