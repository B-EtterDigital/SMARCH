# Wiki Page Contract

A brick wiki page must answer:

1. What does this brick do?
2. When should a user copy it?
3. When should a user avoid it?
4. What is the trust state?
5. What data does it touch?
6. Which security gates are complete?
7. Which models, agents, tools, and humans touched it?
8. What must change when cloned?
9. What checks must run after cloning?

Minimum sections:

- Purpose
- When To Use
- When Not To Use
- Trust
- Gates
- Public API
- Clone Steps
- Model And Human Provenance
- Known Traps

If metadata is missing, say exactly what is missing. Do not invent score, provenance, tests, RLS, env, or security evidence.
