# SMA Enforcer

This README explains the enforcement skill for leases, agent context, validation, promotion, and release checks in a Sweetspot project. Brick owners and agent operators need it whenever they edit a managed module or regenerate shared artifacts. Read it before taking ownership of a brick and before claiming that its gates pass. Remember to acquire the matching lease before edits and release it after verified work.

See [SKILL.md](./SKILL.md) for the lease protocol, validation commands, promotion gate, and continuous-integration setup.
