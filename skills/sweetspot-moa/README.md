# Sweetspot Mixture of Agents ([SMOA](../../docs/GLOSSARY.md#smoa))

This README explains the opt-in orchestration skill that assigns bounded implementation packets to Codex workers while the lead keeps planning and gate authority. Leads, workers, and reviewers need it after the user requests the ensemble mode. Read it before dispatching work, accepting a worker packet, or reviewing a returned result. Remember that workers implement the packet within the strategy and control-plane boundaries set by the lead.

See [SKILL.md](./SKILL.md) for activation, packet structure, review rules, and termination conditions.
